# OS FIRST PRACTISE

## Установка и запуск
Чтобы запустить проект выполните следующие шаги:
1. Клонируйте репозиторий:
2. Перейдите в папу проекта
3. Соберите проект:
```bash
make all
```
4. Для установки библиотеки используйте команду: ```bash
sudo make install``` если потребуется ввести пароль, введите пароль
5. Запустите тесты: ```bash
make test```
6. Если хотите очистить директорию (убрать файлы с расширением .o и .so) выполните команду ```bash
make clean```
