# OS FIRST PRACTISE
Простая динамическая библиотека для шифрования и дешифрования байтов при помощи xor
## Установка и запуск
Чтобы запустить проект выполните следующие шаги:
1. Клонируйте репозиторий:
```bash
git clone https://github.com/vanger2607/MIREA_OS/
```
2. Соберите проект:
```bash
make all
```
3. Для установки библиотеки используйте команду: 
```bash
make install
```
если потребуется ввести пароль, введите пароль
4. Запустите тесты: 
```bash 
make test
```
5. Если хотите очистить директорию (убрать файлы с расширением .o и .so и всопомгательные текстовые файлы) выполните команду 
```bash
 make clean
```
